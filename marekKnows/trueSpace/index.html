TrueSpace

Caligari Corporation (acquired by Microsoft in 2008) created a 3D modeling software application called trueSpace. The VMK in this section will familiarize you with the software's user interface so that you can create 3D models, apply and change material on your models, and work with textures.

Microsoft released the full version of trueSpace for free, and then discontinued the software in 2009. You may still be able to find it for download somewhere online. See Wikipedia for details.

1: Introduction
An introduction to the caligari series of VMK's with instructions on where to download the free version of gameSpace Light from the internet.
2: User Interface
This VMK introduces you to the user interface inside trueSpace v6.6. I show you how to navigate in the 3D scene and how to use the buttons to execute commands and bring up additional menu options.
3: Workflow and Hotkey Setup
This VMK shows you how to setup trueSpace for optimal workflow. Mapping hotkeys for important operations is also done.
4: Using Primitives
I show you how to use the primitive creation tools in this VMK. Primitives are positioned, rotated and scaled. Multiple object selection is also shown.
5: Model Editing
In this VMK I show you how to change the location of an object's pivot point, and how to pull on the model's faces to make interesting shapes.
6: Material Editor
I show you how to use the Material Editor to color an entire object or individual faces. I also show you how to apply transparencies, emissive color, specular color and all other material properties.
7: Texturing Objects
I show you how to use the material editor in this VMK to apply a texture to an object. UV coordinate manipulation is also done using the texture editor.
